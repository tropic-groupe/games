Nos Jeux

Sneaky Snake

Description et contrôle:

Mangez les hamburger pour gagné des point, il y a 12 niveau pour passer d'un niveau à l'autre il vous faut minimum 20 point, vous n'avez que 125 sec pour finir chaque niveau plus vous avez de point plus le serpent et long.

Déplacez-vous avec les touches fléchées (flèche haut, bas, droite, gauche)

Scratch wars épisode 1

Description et contrôle:

Combatter les ennemis, détruiser les porte pour passer aux niveaux suivant.

Déplacez-vous avec les touches fléchées (droite, gauche), touche A pour attaqué, touche flèche du haut pour sauter

Scratch wars épisode 2

Description et contrôle:

Combatter les ennemis, détruiser le Générateur pour ouvrir la porte pour passer aux niveaux suivant.

Déplacez-vous avec les touches fléchées (droite, gauche), touche A pour attaqué, touche flèche du haut pour sauter, ainsi que les touches 1 à 9 pour changer la couleur du sabre
